# Moviearo (THE MOVIE DATABASE API)


![preview moviearo](https://github.com/RenggaRizky/moviearo-movie-tvseries-finder/blob/main/src/assets/images/Moviearo-Menyediakan-jutaan-informasi-perfilman-mancanegara.png)

URL: [Moviearo | Menyediakan jutaan informasi perfilman mancanegara](https://moviearo.netlify.app/ "Moviearo | Menyediakan jutaan informasi perfilman mancanegara")

## Deskripsi
Web yang dibuat untuk menampilkan informasi mengenai film dan serial tv di seluruh dunia, dimana data yang diambil berasal dari The Movie Database API

## Teknologi
Berikut adalah beberapa teknologi yang dipakai untuk membangun web ini
- React
- Tailwind
- Flowbite
- The Movie Database API
